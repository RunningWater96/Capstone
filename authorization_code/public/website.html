Songs Of A Feather

About This Project

When I hear a song, I tend to relate it back to movies or TV shows that I have seen.


This Capstone is somewhat supposed to resemble how my brain thinks in relation to music, movies, and television.


I used the Spotify API for music and the MovieDB API for movies and television shows.


For this whole project, I used javascript and html to create the website you see before you.


For design, I just used CSS, not the sassy kind.
